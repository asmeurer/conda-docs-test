Sample recipes¶

The first two sample recipes, Boost and libtiff, are examples of non-python libraries, meaning, they don’t require python to run or build. This illustrates the flexibility of conda in being able to build things that aren’t python related.

boost is an example of a popular programming library and illustrates the use of selectors in a recipe.

libtiff is another example of a compiled library. This shows how conda is able to apply patches to source directories before building the package.

msgpack, blosc, and cytoolz are examples of Python libraries with extensions.

toolz, sympy, six, and gensim are examples of Python-only libraries.

Gensim works on Python 2, and all the others work on both Python 2 and Python 3.
